Beginner Level

The Six Basic Operations

All of algebra is built on six fundamental operations on real numbers:

1. Addition (\(+\))

Combines two numbers to get their sum.

Example: \(3 + 5 = 8\)
\(3.2 + 1.8 = 5.0\)

2. Subtraction (\(-\))

The inverse operation of addition. Finds the difference between two numbers.

Example: \(8 - 3 = 5\)
\(5 - 7 = -2\)

3. Multiplication (\(\times\) or \(\cdot\))

Repeated addition. Combines numbers by scaling.

Example: \(4 \times 3 = 12\) (add 4 three times)
\(2.5 \times 4 = 10\)

4. Division (\(\div\) or \(/\))

The inverse operation of multiplication. Splits a number into equal parts.

⚠️ CRITICAL RULE: Division by zero is undefined!
We cannot divide by 0 in the real numbers. There is no number \(x\) such that \(0 \cdot x = 5\).

Example: \(12 \div 3 = 4\) (because \(3 \times 4 = 12\))
\(\frac{10}{2} = 5\) (because \(2 \times 5 = 10\))

Invalid: \(\frac{7}{0}\) is undefined ✗

5. Exponentiation (\(x^y\)) or Power

Repeated multiplication. \(a^n\) means multiply \(a\) by itself \(n\) times.

Example: \(2^3 = 2 \times 2 \times 2 = 8\)
\(5^2 = 5 \times 5 = 25\)

6. Logarithm (\(\log\))

The inverse operation of exponentiation. Asks: "What power do I need?"

Example: \(\log_2(8) = 3\) because \(2^3 = 8\)
\(\log_{10}(100) = 2\) because \(10^2 = 100\)

Basic Properties of Addition and Multiplication

Commutativity (Order Doesn't Matter)

For Addition: \(a + b = b + a\)

\(3 + 5 = 8\) and \(5 + 3 = 8\)

For Multiplication: \(a \times b = b \times a\)

\(4 \times 6 = 24\) and \(6 \times 4 = 24\)

NOT for Subtraction or Division:

\(5 - 3 = 2\) but \(3 - 5 = -2\) ✗
\(10 \div 2 = 5\) but \(2 \div 10 = 0.2\) ✗

Associativity (Grouping Doesn't Matter)

For Addition: \((a + b) + c = a + (b + c)\)

\((2 + 3) + 4 = 5 + 4 = 9\) and \(2 + (3 + 4) = 2 + 7 = 9\)

For Multiplication: \((a \times b) \times c = a \times (b \times c)\)

\((2 \times 3) \times 4 = 6 \times 4 = 24\) and \(2 \times (3 \times 4) = 2 \times 12 = 24\)

Identity Elements

Additive Identity: \(a + 0 = a\) (adding 0 doesn't change the number)

\(7 + 0 = 7\), \(0 + (-3) = -3\)

Multiplicative Identity: \(a \times 1 = a\) (multiplying by 1 doesn't change the number)

\(5 \times 1 = 5\), \(1 \times (-2) = -2\)

Inverse Elements

Additive Inverse: \(a + (-a) = 0\) (the inverse undoes the operation)

\(7 + (-7) = 0\), \((-3) + 3 = 0\)

Multiplicative Inverse: \(a \times \frac{1}{a} = 1\) (where \(a \neq 0\))

\(5 \times \frac{1}{5} = 1\), \(\frac{2}{3} \times \frac{3}{2} = 1\)
Note: 0 has NO multiplicative inverse (we cannot multiply by anything to get 1)

Distributivity (Multiplication Distributes Over Addition)

\(a \times (b + c) = (a \times b) + (a \times c)\)

\(3 \times (4 + 5) = 3 \times 9 = 27\)
\((3 \times 4) + (3 \times 5) = 12 + 15 = 27\)

Intermediate Level

Symmetry and Symmetry Properties

Reflexive Property

Every number equals itself: \(a = a\)

\(5 = 5\), \(\pi = \pi\)

Symmetric Property

If \(a = b\), then \(b = a\)

If \(2 + 3 = 5\), then \(5 = 2 + 3\)

Transitive Property

If \(a = b\) and \(b = c\), then \(a = c\)

If \(x + 2 = 5\) and \(5 = 3 + 2\), then \(x + 2 = 3 + 2\)

Cancellation and Inverse Properties

Cancellation Property for Addition

If \(a + c = b + c\), then \(a = b\)

Why? Add the additive inverse of \(c\) to both sides:

If \(x + 5 = 7 + 5\), then \(x = 7\)
Add \(-5\) to both sides: \((x + 5) + (-5) = (7 + 5) + (-5)\)
\(x + 0 = 7 + 0\)
\(x = 7\)

Cancellation Property for Multiplication

If \(a \times c = b \times c\) and \(c \neq 0\), then \(a = b\)

Why? Multiply both sides by the multiplicative inverse of \(c\):

If \(3x = 12\), then \(x = 4\)
Multiply both sides by \(\frac{1}{3}\): \((3x) \times \frac{1}{3} = 12 \times \frac{1}{3}\)
\(x \times 1 = 4\)
\(x = 4\)

Advanced Properties of Exponents

Product Rule: \(a^m \times a^n = a^{m+n}\)

\(2^3 \times 2^2 = 2^{3+2} = 2^5 = 32\)
Check: \((2 \times 2 \times 2) \times (2 \times 2) = 8 \times 4 = 32\) ✓

Quotient Rule: \(\frac{a^m}{a^n} = a^{m-n}\) (where \(a \neq 0\))

\(\frac{2^5}{2^2} = 2^{5-2} = 2^3 = 8\)
Check: \(\frac{32}{4} = 8\) ✓

Power of a Power: \((a^m)^n = a^{m \times n}\)

\((2^3)^2 = 2^{3 \times 2} = 2^6 = 64\)
Check: \((8)^2 = 64\) ✓

Power of a Product: \((a \times b)^n = a^n \times b^n\)

\((2 \times 3)^2 = 2^2 \times 3^2 = 4 \times 9 = 36\)
Check: \((6)^2 = 36\) ✓

Logarithm Properties

Product Rule: \(\log_b(x \times y) = \log_b(x) + \log_b(y)\)

\(\log_2(4 \times 8) = \log_2(4) + \log_2(8) = 2 + 3 = 5\)
Check: \(\log_2(32) = 5\) because \(2^5 = 32\) ✓

Quotient Rule: \(\log_b\left(\frac{x}{y}\right) = \log_b(x) - \log_b(y)\)

\(\log_2\left(\frac{16}{4}\right) = \log_2(16) - \log_2(4) = 4 - 2 = 2\)
Check: \(\log_2(4) = 2\) because \(2^2 = 4\) ✓

Power Rule: \(\log_b(x^n) = n \times \log_b(x)\)

\(\log_2(8^2) = 2 \times \log_2(8) = 2 \times 3 = 6\)
Check: \(\log_2(64) = 6\) because \(2^6 = 64\) ✓

Advanced Level

Axioms of the Real Numbers

The real numbers \(\mathbb{R}\) form a field with the following axioms (fundamental rules):

Field Axioms

Closure Axioms:

  • For any \(a, b \in \mathbb{R}\), we have \(a + b \in \mathbb{R}\) (closed under addition)
  • For any \(a, b \in \mathbb{R}\), we have \(a \times b \in \mathbb{R}\) (closed under multiplication)

Associative Axioms:

  • \((a + b) + c = a + (b + c)\) for all \(a, b, c \in \mathbb{R}\)
  • \((a \times b) \times c = a \times (b \times c)\) for all \(a, b, c \in \mathbb{R}\)

Commutative Axioms:

  • \(a + b = b + a\) for all \(a, b \in \mathbb{R}\)
  • \(a \times b = b \times a\) for all \(a, b \in \mathbb{R}\)

Identity Axioms:

  • There exists \(0 \in \mathbb{R}\) such that \(a + 0 = a\) for all \(a \in \mathbb{R}\)
  • There exists \(1 \in \mathbb{R}\) such that \(a \times 1 = a\) for all \(a \in \mathbb{R}\)
  • \(0 \neq 1\)

Inverse Axioms:

  • For every \(a \in \mathbb{R}\), there exists \(-a \in \mathbb{R}\) such that \(a + (-a) = 0\)
  • For every \(a \in \mathbb{R}\) with \(a \neq 0\), there exists \(a^{-1} \in \mathbb{R}\) such that \(a \times a^{-1} = 1\)

Distributive Axiom:

  • \(a \times (b + c) = (a \times b) + (a \times c)\) for all \(a, b, c \in \mathbb{R}\)

Order Axioms

Beyond being a field, \(\mathbb{R}\) is an ordered field:

  • Totality: For any \(a, b \in \mathbb{R}\), exactly one of these holds: \(a < b\), \(a = b\), or \(a > b\)
  • Transitivity: If \(a < b\) and \(b < c\), then \(a < c\)
  • Compatibility with Addition: If \(a < b\), then \(a + c < b + c\) for all \(c \in \mathbb{R}\)
  • Compatibility with Multiplication: If \(a < b\) and \(c > 0\), then \(a \times c < b \times c\)
  • Sign Flip on Negative Multiplication: If \(a < b\) and \(c < 0\), then \(a \times c > b \times c\)

Completeness Axiom

Unlike the rational numbers \(\mathbb{Q}\), the real numbers have the completeness property:

Every non-empty subset of \(\mathbb{R}\) that is bounded above has a least upper bound (supremum).

Why does this matter? This axiom guarantees that numbers like \(\sqrt{2}\), \(\pi\), and \(e\) actually exist in \(\mathbb{R}\), whereas they don't exist in \(\mathbb{Q}\).

Research Level

Ring and Group Theory

The real numbers form an abelian group under addition and a field under both operations.

Group Structure under Addition

  • \((\mathbb{R}, +)\) is an abelian group
  • Satisfies: Closure, Associativity, Identity (0), Inverses (-a), Commutativity

Field Structure

  • \((\mathbb{R}, +, \times)\) is a field
  • An abelian group under addition
  • \((\mathbb{R} \setminus \{0\}, \times)\) is an abelian group under multiplication
  • Distributive property connects the two operations

Non-Archimedean Fields and Hyperreal Numbers

While \(\mathbb{R}\) is Archimedean (for any \(a, b > 0\), there exists \(n \in \mathbb{N}\) such that \(na > b\)), infinitesimal calculus can be formalized using hyperreal numbers \(^*\mathbb{R}\), which extend \(\mathbb{R}\) with infinitesimals and infinities while preserving field properties.

Transcendental Numbers and Algebraic Closure

The algebraic numbers \(\overline{\mathbb{Q}}\) (roots of polynomial equations with rational coefficients) are a proper subset of \(\mathbb{R}\). Numbers like \(\pi\) and \(e\) are transcendental (not algebraic).

Liouville's Theorem: Transcendental numbers cannot be approximated too well by rational numbers. The measure of transcendence is quantified using Liouville numbers and continued fractions.

Operations on Extended Real Numbers

The extended real numbers \(\overline{\mathbb{R}} = \mathbb{R} \cup \{-\infty, +\infty\}\) extend operations but with restrictions:

  • \(a + \infty = +\infty\) for \(a > -\infty\)
  • \(a \times 0 = 0\) is indeterminate when \(a\) involves \(\infty\)
  • \(\frac{\infty}{\infty}\) and \(\frac{0}{0}\) remain undefined

Abstract Algebra: Homomorphisms and Isomorphisms

Operations on \(\mathbb{R}\) can be studied through field homomorphisms—functions that preserve the algebraic structure.

A field homomorphism \(\phi: \mathbb{R} \to \mathbb{R}\) satisfies:

  • \(\phi(a + b) = \phi(a) + \phi(b)\)
  • \(\phi(a \times b) = \phi(a) \times \phi(b)\)
  • \(\phi(1) = 1\)

The only field automorphism of \(\mathbb{R}\) is the identity function (this follows from the order structure).
