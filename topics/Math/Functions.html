Beginner Level

What is a Function?

A function is a relationship between inputs and outputs where each input has exactly one output.

Notation: \(f: A \rightarrow B\) means \(f\) is a function from set \(A\) to set \(B\).

For an input \(x\), we write \(f(x)\) for the output (the image of \(x\)).

  • \(A\) is the domain (all possible inputs)
  • \(B\) is the codomain (set containing all possible outputs)
  • The range (or image) is the set of actual outputs: \(\{f(x) \mid x \in A\}\)

Function Notation and Representation

Algebraic Notation:

\[f(x) = x^2 + 2x + 1\]

Ordered Pairs:

\[f = \{(1, 3), (2, 5), (3, 7)\}\]

Common Functions

Linear Functions

Intuition: A linear function models any quantity that changes at a constant rate: every extra unit of input adds the same amount to the output. Think of a taxi meter — a fixed starting fee, plus a fixed price for every kilometer.

Formal definition:

\[f(x) = mx + b\]

  • \(m\) is the slope: the change in \(f(x)\) per unit change in \(x\) (rise over run)
  • \(b\) is the y-intercept: the value \(f(0)\), where the line crosses the y-axis
  • The graph is a straight line

Interactive plot: drag the knobs for \(m\) and \(b\) and the range bars. The green dot marks the y-intercept; the triangle shows run = 1, rise = \(m\).

Step-by-step: Find the line through \((1, 5)\) and \((3, 11)\)

Step 1: Compute the slope
Why? The slope is the rate of change between any two points on the line.
\[m = \frac{y_2 - y_1}{x_2 - x_1} = \frac{11 - 5}{3 - 1} = \frac{6}{2} = 3\]
Step 2: Find the intercept b
Why? We know \(f(x) = 3x + b\); substitute either known point to pin down \(b\). Using \((1, 5)\):
\[5 = 3(1) + b \Rightarrow b = 2\]
Step 3: Write and verify
\[f(x) = 3x + 2\] Check with the other point: \(f(3) = 3(3) + 2 = 11\) ✓
Real-World Applications:
  • Taxi fare (worked): base fare €3.50 plus €1.20/km: \(C(d) = 1.2d + 3.5\). A 8 km ride costs \(C(8) = 1.2(8) + 3.5 = 13.10\) euros.
  • Temperature conversion: \(F(c) = \tfrac{9}{5}c + 32\) — slope \(9/5\), intercept 32.
  • Ohm's law: \(V = RI\) — voltage is linear in current with slope \(R\) and intercept 0.
  • Linear depreciation: asset value \(V(t) = V_0 - kt\) (negative slope).

Exercises:

  1. Find the linear function through \((-2, 1)\) and \((4, 13)\).
    Solution \(m = \frac{13 - 1}{4 - (-2)} = \frac{12}{6} = 2\). Then \(1 = 2(-2) + b \Rightarrow b = 5\). So \(f(x) = 2x + 5\). Check: \(f(4) = 13\) ✓
  2. A phone plan costs $20/month plus $0.05 per minute. Write the cost function and find the cost of 300 minutes.
    Solution \(C(t) = 0.05t + 20\). \(C(300) = 0.05(300) + 20 = 15 + 20 = \$35\).

Quadratic Functions

Intuition: A quadratic function models quantities whose rate of change itself changes at a constant rate — like a thrown ball, whose speed changes steadily under gravity. Its graph is a parabola: perfectly symmetric around its turning point.

Formal definition:

\[f(x) = ax^2 + bx + c \quad (a \neq 0)\]

  • Graph is a parabola, opening up if \(a > 0\), down if \(a < 0\)
  • Vertex (turning point) at \(x_v = -\frac{b}{2a}\)
  • Discriminant \(\Delta = b^2 - 4ac\) counts real roots: two if \(\Delta > 0\), one if \(\Delta = 0\), none if \(\Delta < 0\)

Interactive plot: the green dot is the vertex; root dots appear when \(\Delta \ge 0\). Slide \(a\) through 0 to see the parabola degenerate into a line.

Step-by-step: Completing the square for \(f(x) = x^2 - 4x + 3\)

Step 1: Group the \(x\)-terms
Why? We want to force \(x^2 - 4x\) to look like a perfect square \((x - h)^2\).
\[f(x) = (x^2 - 4x) + 3\]
Step 2: Add and subtract \((b/2)^2\)
Why? \((x-2)^2 = x^2 - 4x + 4\), so \(x^2 - 4x\) is a perfect square that is missing the constant \(4 = (4/2)^2\). We add it and immediately subtract it, changing nothing:
\[f(x) = (x^2 - 4x + 4) - 4 + 3\]
Step 3: Factor the perfect square
\[f(x) = (x - 2)^2 - 1\]
Step 4: Read off the vertex
Why? \((x-2)^2 \geq 0\) always, and equals 0 exactly at \(x = 2\). So the minimum is at \((2, -1)\) — matching \(x_v = -b/2a = 4/2 = 2\). ✓

Geometric Picture (algebra tiles): \(x^2 + bx\) is literally a square of side \(x\) plus a \(b \times x\) strip. Cut the strip in half and hang the two halves on the right and bottom edges of the square: you get an almost-square of side \(x + b/2\), missing only its tiny \((b/2)^2\) corner. "Completing the square" means paying that missing corner — that is exactly the \(+\,(b/2)^2\) in Step 2.

Real-World Applications:
  • Projectile motion (worked): a ball thrown up at \(v_0 = 20\) m/s from height 1.5 m: \(h(t) = -4.9t^2 + 20t + 1.5\). It lands when \(h(t) = 0\): \(t = \frac{-20 - \sqrt{400 + 4(4.9)(1.5)}}{2(-4.9)} \approx 4.156\) s. Peak at \(t_v = \frac{20}{9.8} \approx 2.04\) s, height \(h(2.04) \approx 21.9\) m.
  • Profit maximization: revenue \(R(p) = p(1000 - 50p)\) is a downward parabola — the best price sits at the vertex.
  • Parabolic reflectors: satellite dishes and headlights use the parabola's focusing property.

Exercises:

  1. Complete the square for \(f(x) = 2x^2 + 8x + 5\) and give the vertex.
    Solution \(f(x) = 2(x^2 + 4x) + 5 = 2(x^2 + 4x + 4) - 8 + 5 = 2(x+2)^2 - 3\). Vertex \((-2, -3)\).
  2. For which values of \(k\) does \(x^2 + kx + 9 = 0\) have exactly one real root?
    Solution One root ⟺ \(\Delta = k^2 - 36 = 0\) ⟺ \(k = \pm 6\).

Polynomial Functions

Intuition: A polynomial is a sum of power terms — the workhorse of applied mathematics, because polynomials can bend as many times as their degree allows while staying smooth and easy to compute.

Formal definition:

\[f(x) = a_nx^n + a_{n-1}x^{n-1} + \dots + a_1x + a_0, \quad a_n \neq 0\]

where \(n\) is the degree (highest power). A degree-\(n\) polynomial has at most \(n\) real roots and at most \(n-1\) turning points.

Interactive plot: change the degree and watch the end behavior flip with the parity of \(n\) and the sign of the leading coefficient (annotated in the corners).

Step-by-step: End behavior rules
For large \(|x|\), the leading term \(a_nx^n\) dominates everything else. Four cases:

Degree \(n\)\(a_n > 0\)\(a_n < 0\)
evenboth ends ↑ (like \(x^2\))both ends ↓
oddleft ↓, right ↑ (like \(x^3\))left ↑, right ↓
Step-by-step: Factor \(f(x) = x^3 - 2x^2 - 5x + 6\) completely

Step 1: Hunt for a rational root
Why? By the rational root theorem, any rational root divides the constant 6: try \(\pm1, \pm2, \pm3, \pm6\).
\(f(1) = 1 - 2 - 5 + 6 = 0\) ✓ — so \((x - 1)\) is a factor.

Step 2: Divide out the factor
Why? Polynomial division reduces the degree so we can finish with the quadratic formula.
\[x^3 - 2x^2 - 5x + 6 = (x - 1)(x^2 - x - 6)\]
Step 3: Factor the quadratic
\(x^2 - x - 6 = (x - 3)(x + 2)\)

Result: \(f(x) = (x - 1)(x - 3)(x + 2)\); roots \(x = 1, 3, -2\).
Real-World Applications:
  • Open-box volume (worked setup): cut squares of side \(x\) from the corners of a 20×20 cm sheet and fold: \(V(x) = x(20 - 2x)^2\), a cubic. Reading its graph (or later, calculus) gives the best cut \(x = 10/3\) cm.
  • Bézier curves: every font glyph and vector graphic is stitched from cubic polynomials.
  • Polynomial regression: fitting curved trends in data.

Exercises:

  1. What is the maximum number of turning points of a degree-5 polynomial, and why?
    Solution 4. Turning points happen where \(f'(x) = 0\); the derivative has degree 4, hence at most 4 real roots.
  2. Factor \(f(x) = x^3 - 7x + 6\).
    Solution \(f(1) = 0\), divide: \(x^3 - 7x + 6 = (x-1)(x^2 + x - 6) = (x-1)(x+3)(x-2)\).

Rational Functions

Intuition: A rational function is a ratio of polynomials. Ratios blow up where the denominator vanishes — that is where the graph tears apart into branches separated by asymptotes.

Formal definition:

\[f(x) = \frac{p(x)}{q(x)}, \quad q(x) \neq 0\]

  • Vertical asymptote at every root of \(q\) that is not cancelled by \(p\)
  • Horizontal asymptote from comparing degrees of \(p\) and \(q\)
  • Hole (removable point) where a factor cancels

Interactive plot: the Möbius family \(f(x) = \frac{ax + b}{cx + d}\). Dashed lines are the vertical asymptote \(x = -d/c\) and horizontal asymptote \(y = a/c\).

Step-by-step: All asymptotes and holes of \(f(x) = \dfrac{x^2 - 1}{x^2 - 4}\)

Step 1: Factor top and bottom
Why? Asymptotes and holes are decided by the factored form.
\[f(x) = \frac{(x-1)(x+1)}{(x-2)(x+2)}\]
Step 2: Vertical asymptotes
Why? The denominator vanishes at \(x = 2\) and \(x = -2\); neither factor cancels with the numerator, so both are true vertical asymptotes (not holes).

Step 3: Horizontal asymptote
Why? Degrees of numerator and denominator are equal (both 2), so \(y = \) ratio of leading coefficients \(= 1/1 = 1\).

Step 4: Holes
No common factors cancelled, so there are no holes. (Compare: \(\frac{(x-1)(x+1)}{(x-1)(x+2)}\) would have a hole at \(x = 1\) instead of an asymptote.)
Real-World Applications:
  • Average cost (worked): producing \(x\) items costs \(C(x) = 500 + 2x\); the average cost per item \(A(x) = \frac{500 + 2x}{x} = \frac{500}{x} + 2\) → horizontal asymptote \(y = 2\): mass production approaches the marginal cost.
  • Thin lens equation: \(\frac{1}{f} = \frac{1}{u} + \frac{1}{v}\) rearranges to the rational \(v(u) = \frac{uf}{u - f}\), with a vertical asymptote at the focal length.
  • Michaelis–Menten kinetics: reaction rate \(v = \frac{V_{max}[S]}{K_m + [S]}\) — saturation is the horizontal asymptote.

Exercises:

  1. Find all asymptotes and holes of \(f(x) = \dfrac{x^2 - 9}{x - 3}\).
    Solution \(f(x) = \frac{(x-3)(x+3)}{x-3} = x + 3\) for \(x \neq 3\): no asymptotes, one hole at \((3, 6)\).
  2. For \(f(x) = \dfrac{2x + 1}{x - 4}\): asymptotes, and the value \(f\) can never take.
    Solution Vertical \(x = 4\); horizontal \(y = 2\). \(f(x) = 2\) would need \(2x + 1 = 2x - 8\), impossible — so \(f\) never equals 2 (exactly the horizontal asymptote).

Function Properties: Even/Odd & Monotonicity

Intuition: Symmetry is free information — if you know an even or odd function on the right half of the axis, you know it everywhere.

Even and Odd Functions

A function \(f\) is even if \(f(-x) = f(x)\) — symmetric about the y-axis (mirror).

A function \(f\) is odd if \(f(-x) = -f(x)\) — symmetric about the origin (180° rotation).

Interactive plot: the solid curve is \(f(x)\); the dashed green curve is \(f(-x)\). If dashed lands exactly on solid → even. If dashed is the mirror image through the x-axis → odd. Anything else → neither.

Step-by-step: Prove \(f(x) = x^3\) is odd

Step 1: Write \(f(-x)\) and expand
\[f(-x) = (-x)^3 = (-1)^3 x^3 = -x^3\]
Step 2: Compare with \(-f(x)\)
\(-f(x) = -x^3\). They agree for every \(x\), so \(f(-x) = -f(x)\): \(f\) is odd. ∎
Step-by-step: Show \(g(x) = x^2 + x\) is neither

Step 1: \(g(-x) = x^2 - x\).
Step 2: Test evenness at \(x = 1\): \(g(-1) = 0 \neq 2 = g(1)\). Not even.
Step 3: Test oddness at \(x = 1\): \(g(-1) = 0 \neq -2 = -g(1)\). Not odd. ∎

Monotonicity

A function \(f\) is:

  • Increasing if \(x_1 < x_2 \Rightarrow f(x_1) < f(x_2)\)
  • Decreasing if \(x_1 < x_2 \Rightarrow f(x_1) > f(x_2)\)

Most functions are monotonic only piecewise: \(x^2\) decreases on \((-\infty, 0]\) and increases on \([0, \infty)\). Monotonic functions are exactly the ones that pass the horizontal line test everywhere — hence invertible (see Inverse Functions).

Real-World Applications:
  • Fourier analysis: every signal splits uniquely into an even part \(\frac{f(x)+f(-x)}{2}\) and an odd part \(\frac{f(x)-f(-x)}{2}\) — cosines carry the even part, sines the odd part.
  • Integration shortcut (worked): \(\int_{-a}^{a} x^3\cos(x^2)\,dx = 0\) instantly, because the integrand is odd — no computation needed.
  • Monotonic response curves: dose-response in pharmacology and cumulative distribution functions in statistics are monotone by construction.

Exercises:

  1. Classify \(f(x) = \dfrac{x}{x^2 + 1}\) as even, odd, or neither.
    Solution \(f(-x) = \frac{-x}{x^2+1} = -f(x)\) → odd.
  2. Split \(f(x) = e^x\) into its even and odd parts.
    Solution Even: \(\frac{e^x + e^{-x}}{2} = \cosh x\). Odd: \(\frac{e^x - e^{-x}}{2} = \sinh x\). Indeed \(e^x = \cosh x + \sinh x\).

Special Functions

Exponential Functions

Intuition: An exponential function models anything that grows (or decays) in proportion to its current size — populations, invested money, radioactive atoms. Linear growth adds a fixed amount per step; exponential growth multiplies by a fixed factor per step.

Formal definition:

\[f(x) = C \cdot a^x \quad (a > 0,\ a \neq 1)\]

  • Domain: all real numbers; Range: \((0, \infty)\) for \(C > 0\)
  • Passes through \((0, C)\); base \(a > 1\) grows, \(0 < a < 1\) decays
  • \(a = 1\) is degenerate: the constant function \(C\)

Interactive plot: toggle the log-scale checkbox — on a logarithmic y-axis every exponential becomes a straight line (that's how scientists spot exponential growth in data).

Step-by-step: From compound interest to the number \(e\)

Step 1: Compound once a year
€1 at 100% interest becomes \((1 + 1)^1 = 2\).

Step 2: Compound \(n\) times a year
Why? Each period pays \(1/n\) of the rate, \(n\) times: \(\left(1 + \frac{1}{n}\right)^n\).

Step 3: Let \(n\) grow

Step 4: The limit is the natural base \[e = \lim_{n \to \infty}\left(1 + \frac{1}{n}\right)^n \approx 2.71828\]
Real-World Applications:
  • Carbon-14 dating (worked): \(N(t) = N_0 \cdot (1/2)^{t/5730}\). A bone with 25% of its original C-14 satisfies \((1/2)^{t/5730} = 1/4 = (1/2)^2\), so \(t = 2 \times 5730 = 11\,460\) years.
  • Compound interest (worked): €1000 at 5% for 10 years: \(1000(1.05)^{10} \approx 1628.89\) euros.
  • Population growth / epidemics: early-phase spread multiplies per generation.
  • Moore's law: transistor counts doubling ≈ every 2 years — a straight line on the widget's log scale.

Exercises:

  1. A bacteria culture triples every 4 hours starting from 500 cells. Write \(N(t)\) and evaluate at \(t = 12\) h.
    Solution \(N(t) = 500 \cdot 3^{t/4}\); \(N(12) = 500 \cdot 3^3 = 13\,500\) cells.
  2. Which is larger for big \(x\): \(f(x) = x^{100}\) or \(g(x) = 1.01^x\)? Why?
    Solution \(g\) eventually wins: any exponential with base \(> 1\) outgrows every polynomial, since \(\frac{g(x)}{f(x)} = e^{x\ln 1.01 - 100\ln x} \to \infty\).

Logarithmic Functions

Intuition: A logarithm answers the question "how many times must I multiply by \(a\) to reach \(x\)?" It compresses huge ranges into small ones — which is why earthquakes, sound, and acidity are all measured logarithmically.

Formal definition:

\[f(x) = \log_a(x) \quad (a > 0,\ a \neq 1)\]

  • Domain: \((0, \infty)\); Range: all real numbers
  • Inverse of the exponential: \(y = a^x \iff x = \log_a(y)\)

The three log laws (each is an exponent law in disguise):

  • \(\log_a(xy) = \log_a x + \log_a y\) — because \(a^{m+n} = a^m a^n\)
  • \(\log_a(x/y) = \log_a x - \log_a y\) — because \(a^{m-n} = a^m / a^n\)
  • \(\log_a(x^k) = k \log_a x\) — because \((a^m)^k = a^{mk}\)

Interactive plot: \(\log_a x\) (solid), its inverse \(a^x\) (dashed green), and the mirror line \(y = x\) (dotted) — inverse pairs are always reflections across it.

Step-by-step: Solve \(3 \cdot 2^{x+1} = 24\)

Step 1: Isolate the exponential
Why? Logs can only "undo" a bare exponential.
\[2^{x+1} = 8\]
Step 2: Take \(\log_2\) of both sides
Why? \(\log_2\) is the inverse of \(2^x\), so it frees the exponent (law 3 with \(k = x+1\)):
\[x + 1 = \log_2 8 = 3\]
Step 3: Solve \[x = 2\] Verification: \(3 \cdot 2^{3} = 24\) ✓
Real-World Applications:
  • Richter scale (worked): magnitude difference 7 − 5 = 2 means \(10^2 = 100\) times the shaking amplitude and \(\approx 10^{1.5 \times 2} = 1000\) times the energy.
  • pH (worked): \(\text{pH} = -\log_{10}[\text{H}^+]\). Coffee at pH 5 has \(10^{-5}\) mol/L — 100× more acidic than water at pH 7.
  • Binary search: halving a million-entry list takes \(\log_2 10^6 \approx 20\) steps.
  • Decibels & information entropy: both are logarithms of ratios.

Exercises:

  1. Solve \(\log_3(x) + \log_3(x - 2) = 1\).
    Solution Law 1: \(\log_3 x(x-2) = 1 \Rightarrow x^2 - 2x = 3 \Rightarrow (x-3)(x+1) = 0\). Only \(x = 3\) keeps both logs defined (\(x > 2\)). ✓
  2. How many decimal digits does \(2^{100}\) have?
    Solution Digits \(= \lfloor 100 \log_{10} 2 \rfloor + 1 = \lfloor 30.103 \rfloor + 1 = 31\).

Trigonometric Functions

Intuition: Trigonometric functions are what circular motion looks like from the side. A point going around a circle, watched edge-on, bobs up and down as a sine wave — that single fact makes \(\sin\) and \(\cos\) the language of every wave and vibration in physics.

The six functions: \(\sin(x), \cos(x), \tan(x) = \frac{\sin x}{\cos x}, \cot(x), \sec(x), \csc(x)\)

Fundamental identity (Pythagoras on the unit circle):

\[\sin^2(x) + \cos^2(x) = 1\]

Interactive animation: press Play — the point circles the unit circle (left) while its height traces the sine wave (right). Knobs control \(A\sin(\omega x + \varphi)\).

Step-by-step: Exact values from the unit circle

θ = π/6 (30°): half an equilateral triangle of side 1 → legs \(\frac12\) and \(\frac{\sqrt3}{2}\): \(\sin\frac{\pi}{6} = \frac12\), \(\cos\frac{\pi}{6} = \frac{\sqrt3}{2}\).

θ = π/4 (45°): isosceles right triangle with hypotenuse 1 → legs \(\frac{1}{\sqrt2}\): \(\sin\frac{\pi}{4} = \cos\frac{\pi}{4} = \frac{\sqrt2}{2}\).

θ = π/3 (60°): the π/6 triangle with legs swapped: \(\sin\frac{\pi}{3} = \frac{\sqrt3}{2}\), \(\cos\frac{\pi}{3} = \frac12\).
Real-World Applications:
  • Concert pitch A440 (worked): the tuning-fork tone is \(p(t) = A\sin(2\pi \cdot 440\, t)\) — 440 full circles of the widget per second.
  • AC mains voltage: \(V(t) = 325\sin(2\pi \cdot 50\,t)\) volts in Europe (230 V RMS).
  • Tides, seasons, springs: any system pulled back toward equilibrium proportionally to displacement oscillates sinusoidally (simple harmonic motion).

Exercises:

  1. Find amplitude, period, and phase shift of \(f(x) = 3\sin(2x - \frac{\pi}{2})\).
    Solution \(A = 3\); period \(= \frac{2\pi}{2} = \pi\); phase shift \(= \frac{\pi/2}{2} = \frac{\pi}{4}\) to the right.
  2. Using the unit circle, explain why \(\sin(\pi - \theta) = \sin\theta\).
    Solution The points at angles \(\theta\) and \(\pi - \theta\) are mirror images across the y-axis: same height (same sine), opposite \(x\) (cosine flips sign).

Inverse Functions

Intuition: An inverse function runs a process backwards: if \(f\) converts euros to dollars, \(f^{-1}\) converts dollars back to euros. Only processes that never merge two inputs into one output can be reversed.

Formal definition: If \(f: A \rightarrow B\) is a bijection (one-to-one and onto), then \(f^{-1}: B \rightarrow A\) is the inverse function satisfying:

\[f(f^{-1}(x)) = x \quad \text{and} \quad f^{-1}(f(x)) = x\]

Graphically: the graph of \(f^{-1}\) is the graph of \(f\) reflected across the line \(y = x\) — swapping input and output swaps the axes.

Step-by-step: Invert \(f(x) = 2x + 3\)

Step 1: Write \(y = f(x)\) and swap \(x \leftrightarrow y\)
Why? The inverse exchanges the roles of input and output.
\[x = 2y + 3\]
Step 2: Solve for \(y\)
\[y = \frac{x - 3}{2} \quad\Rightarrow\quad f^{-1}(x) = \frac{x - 3}{2}\]
Step 3: Verify BOTH compositions
\(f(f^{-1}(x)) = 2 \cdot \frac{x-3}{2} + 3 = x\) ✓ and \(f^{-1}(f(x)) = \frac{(2x+3) - 3}{2} = x\) ✓
Why \(x^2\) needs a domain restriction: a horizontal line \(y = 4\) hits the parabola twice (\(x = \pm 2\)) — it fails the horizontal line test, so no inverse exists on all of \(\mathbb{R}\). Restrict to \([0, \infty)\) and the inverse is \(\sqrt{x}\).
Real-World Applications:
  • Decryption: decoding is exactly the inverse function of encoding — RSA's security rests on inverses that are easy one way, hard the other.
  • Unit conversions: \(F(c) = \tfrac95 c + 32\) has inverse \(C(f) = \tfrac59(f - 32)\).
  • log ↔ exp: see Logarithmic Functions — the widget there shows the same mirror picture.

Exercises:

  1. Find \(f^{-1}\) for \(f(x) = \dfrac{x}{x + 1}\) (for \(x \neq -1\)).
    Solution \(x = \frac{y}{y+1} \Rightarrow x(y+1) = y \Rightarrow y(x - 1) = -x \Rightarrow f^{-1}(x) = \frac{x}{1 - x}\), \(x \neq 1\).
  2. Is \(f(x) = x^3 - x\) invertible on \(\mathbb{R}\)?
    Solution No: \(f(0) = f(1) = f(-1) = 0\) — three inputs share one output, failing the horizontal line test. (It becomes invertible on \([1/\sqrt3, \infty)\), where it is strictly increasing.)

Function Composition & Transformations

Intuition: Composition chains machines: the output of \(g\) feeds straight into \(f\). Transformations are the special case of composing with simple shift/stretch machines — which is how one template graph generates a whole family.

Composition

\[(f \circ g)(x) = f(g(x)) \quad \text{(apply } g \text{ first, then } f\text{)}\]

Step-by-step: \(f(x) = x^2\), \(g(x) = x + 1\) — both orders

Order 1: \((f \circ g)(x)\)
Feed \(g\)'s output into \(f\): \(f(g(x)) = f(x+1) = (x+1)^2 = x^2 + 2x + 1\).
At \(x = 2\): \(g(2) = 3\), then \(f(3) = 9\).

Order 2: \((g \circ f)(x)\)
Feed \(f\)'s output into \(g\): \(g(f(x)) = g(x^2) = x^2 + 1\).
At \(x = 2\): \(f(2) = 4\), then \(g(4) = 5\).

Conclusion: \(9 \neq 5\) — composition is not commutative: \(f \circ g \neq g \circ f\) in general.

Transformations

Every transformed graph is \(a \cdot f(b(x - h)) + k\):

  • \(k\): vertical shift (up if \(k > 0\))
  • \(h\): horizontal shift (right if \(h > 0\))
  • \(a\): vertical stretch (\(|a| > 1\)) / compression (\(|a| < 1\)); \(a < 0\) reflects across the x-axis
  • \(b\): horizontal compression (\(|b| > 1\)) / stretch (\(|b| < 1\)); \(b < 0\) reflects across the y-axis

Interactive plot: base curve dashed, transformed curve solid. Watch the readout rebuild the formula as you turn the knobs.

Step-by-step: Build \(g(x) = -2f(x - 3) + 1\) from \(f\) — order matters!

Step 1: Horizontal shift — \(f(x - 3)\): slide the graph 3 right. (Do shifts inside the argument first.)
Step 2: Vertical stretch — \(2f(x-3)\): double every height.
Step 3: Reflection — \(-2f(x-3)\): flip across the x-axis.
Step 4: Vertical shift — \(-2f(x-3) + 1\): raise everything by 1.

Why this order? Steps 2–4 act on the output; doing the \(+1\) before the \(\times(-2)\) would give \(-2(f + 1) = -2f - 2\), a different function.
Real-World Applications:
  • Computer graphics: every sprite you see is scaled (\(a, b\)), translated (\(h, k\)), and flipped (signs) — the widget's four knobs are a 2D graphics pipeline in miniature.
  • Signal processing: \(f(t - h)\) is a time delay; \(f(bt)\) is playback speed.
  • Currency chains (composition): EUR→USD→JPY is \((g \circ f)\); the order of conversion never commutes with fees involved.

Exercises:

  1. Given \(f(x) = \sqrt{x}\) and \(g(x) = 3x + 1\), compute \((f \circ g)(5)\) and \((g \circ f)(5)\).
    Solution \((f \circ g)(5) = f(16) = 4\). \((g \circ f)(5) = g(\sqrt5) = 3\sqrt5 + 1 \approx 7.708\).
  2. Describe how the graph of \(y = \frac12|x + 2| - 3\) is built from \(y = |x|\).
    Solution Shift 2 left (\(x + 2 = x - (-2)\)), compress vertically by \(\frac12\), then shift 3 down. Vertex moves \((0,0) \to (-2, -3)\).

Advanced Level

Continuous Functions

Intuition: A continuous function is one you can draw without lifting the pen: nearby inputs give nearby outputs, with no jumps or teleports.

Formal definition: \(f: \mathbb{R} \rightarrow \mathbb{R}\) is continuous at \(c\) if

\[\lim_{x \to c} f(x) = f(c)\]

Equivalent (ε-δ) definition: for every \(\varepsilon > 0\) there exists \(\delta > 0\) such that

\[|x - c| < \delta \;\Rightarrow\; |f(x) - f(c)| < \varepsilon\]

Geometric meaning (this is what the widget below shows): whatever horizontal ε-tube you draw around the height \(f(c)\), you can always find a vertical δ-window around \(c\) so narrow that the whole piece of graph inside the window stays inside the tube. Continuity at \(c\) means the challenge "here's my ε!" can always be answered with a working δ.

Interactive: ε-δ on the continuous \(f(x) = x^2\). Shrink ε and watch δ shrink too — but a valid δ always exists.

Interactive: ε-δ FAILING on a jump. The step function \(f(x) = x\) for \(x < 1\), \(x + 2\) for \(x \ge 1\). Put \(c = 1\) and pick \(\varepsilon < 2\): no δ can keep the graph inside the tube — the jump always escapes.

Step-by-step ε-δ proof: \(f(x) = 3x + 1\) is continuous at \(c = 2\)

Step 1: Write what must be controlled
We must make \(|f(x) - f(2)| = |3x + 1 - 7| = 3|x - 2|\) smaller than any given \(\varepsilon\).

Step 2: Choose δ
Why? \(3|x - 2| < \varepsilon\) exactly when \(|x - 2| < \varepsilon/3\). So choose \(\delta = \varepsilon/3\).

Step 3: Verify the implication
If \(|x - 2| < \delta = \varepsilon/3\), then \(|f(x) - f(2)| = 3|x - 2| < 3 \cdot \varepsilon/3 = \varepsilon\). ∎

(The widget's readout for a linear function shows exactly this ratio: δ is always ε divided by the slope.)
The three types of discontinuity:
  • Removable: \(\frac{x^2 - 1}{x - 1}\) at \(x = 1\) — a single missing point (hole); redefining one value fixes it.
  • Jump: the widget's step function at \(x = 1\) — left and right limits exist but disagree.
  • Infinite: \(1/x\) at \(x = 0\) — the function blows up.

Two theorems that need continuity (both visible in the widgets):

  • Intermediate Value Theorem (IVT): a continuous \(f\) on \([a, b]\) takes every value between \(f(a)\) and \(f(b)\) — the unbroken pen stroke must cross every horizontal line between its endpoints. The jump widget shows how a discontinuity lets the graph skip values 1 to 3 entirely.
  • Extreme Value Theorem (EVT): a continuous \(f\) on a closed \([a, b]\) attains a maximum and minimum.
Real-World Applications:
  • Root-finding: the bisection algorithm is IVT made mechanical — it only works because continuity forbids skipping zero (see Numerical Methods).
  • Physical signals: temperature, pressure, position evolve continuously — sensors exploit this to interpolate between samples.
  • Thermostats & switches: engineered jump discontinuities: output snaps between states at a threshold.

Exercises:

  1. Prove with ε-δ that \(f(x) = 5x - 4\) is continuous at \(c = 1\).
    Solution \(|f(x) - f(1)| = |5x - 4 - 1| = 5|x - 1|\). Given \(\varepsilon > 0\) choose \(\delta = \varepsilon/5\): then \(|x-1| < \delta \Rightarrow |f(x)-f(1)| = 5|x-1| < \varepsilon\). ∎
  2. Classify the discontinuity of \(f(x) = \frac{|x|}{x}\) at \(x = 0\).
    Solution Left limit \(-1\), right limit \(+1\): both exist but differ → jump discontinuity (of size 2).

Differentiable Functions

Intuition: A function is differentiable at a point if, zoomed in far enough, its graph looks like a straight line there. The slope of that line is the derivative. Corners and kinks (like \(|x|\) at 0) never straighten out, no matter the zoom — they are not differentiable.

Formal definition: \(f\) is differentiable at \(c\) if the limit

\[f'(c) = \lim_{h \to 0} \frac{f(c + h) - f(c)}{h}\]

exists (finite). In ε-δ form: \(\exists L \in \mathbb{R}\) such that \(\forall \varepsilon > 0\ \exists \delta > 0:\)

\[0 < |h| < \delta \;\Rightarrow\; \left|\frac{f(c+h) - f(c)}{h} - L\right| < \varepsilon\]

Interactive: secant → tangent on \(f(x) = x^2/2\). Press Play: the point \(b\) slides toward \(a\), the secant collapses onto the tangent, and the difference quotient converges to \(f'(a) = a\).

Interactive: where differentiability FAILS — \(f(x) = |x|\) at 0. The secant from the left has slope −1, from the right +1. They never agree, so no tangent exists at the corner.

Worked examples from the limit definition

Example 1: \(f(x) = x^2\) at a general point \(c\)

Step 1: Form the difference quotient: \[\frac{f(c+h) - f(c)}{h} = \frac{(c+h)^2 - c^2}{h} = \frac{2ch + h^2}{h}\] Step 2: For \(h \neq 0\), cancel \(h\): \(= 2c + h\).
Step 3: Take the limit: \(f'(c) = \lim_{h \to 0}(2c + h) = 2c\). ∎
Example 2: \(f(x) = \frac{1}{x}\) (for \(c \neq 0\))

Step 1: \[\frac{\frac{1}{c+h} - \frac{1}{c}}{h} = \frac{\frac{c - (c+h)}{c(c+h)}}{h} = \frac{-h}{h \, c(c+h)}\] Step 2: Cancel \(h\): \(= \frac{-1}{c(c+h)}\).
Step 3: Limit: \(f'(c) = -\frac{1}{c^2}\). ∎
Example 3: \(f(x) = \sqrt{x}\) (for \(c > 0\)) — the conjugate trick

Step 1: Multiply by the conjugate to remove the root difference: \[\frac{\sqrt{c+h} - \sqrt{c}}{h} \cdot \frac{\sqrt{c+h} + \sqrt{c}}{\sqrt{c+h} + \sqrt{c}} = \frac{(c+h) - c}{h(\sqrt{c+h} + \sqrt{c})}\] Step 2: Cancel \(h\): \(= \frac{1}{\sqrt{c+h} + \sqrt{c}}\).
Step 3: Limit: \(f'(c) = \frac{1}{2\sqrt{c}}\). ∎
Example 4 (failure): \(f(x) = |x|\) at \(c = 0\)

Right limit: \(\lim_{h \to 0^+} \frac{|h| - 0}{h} = \lim_{h \to 0^+} \frac{h}{h} = +1\)
Left limit: \(\lim_{h \to 0^-} \frac{|h| - 0}{h} = \lim_{h \to 0^-} \frac{-h}{h} = -1\)
One-sided limits disagree (\(+1 \neq -1\)) → the two-sided limit does not exist → \(|x|\) is not differentiable at 0 (exactly what the widget shows). ∎

Theorem: differentiable ⇒ continuous

Proof sketch: if \(f'(c)\) exists, then \[\lim_{h \to 0}\big(f(c+h) - f(c)\big) = \lim_{h \to 0} \frac{f(c+h) - f(c)}{h} \cdot h = f'(c) \cdot 0 = 0,\] which is precisely continuity at \(c\). ∎

The converse is FALSE: \(|x|\) is continuous everywhere but not differentiable at 0. (Extreme counterexample: the Weierstrass function is continuous everywhere and differentiable nowhere.)
Real-World Applications:
  • Velocity & acceleration: \(v(t) = s'(t)\), \(a(t) = v'(t)\) — the difference quotient is literally average speed over a shrinking stopwatch interval.
  • Marginal cost: \(C'(q)\) = cost of producing one more unit.
  • Newton's method: uses tangent lines to find roots fast (see Numerical Methods).
  • Gradient descent in ML: training a neural network is following derivatives downhill.

Exercises:

  1. Compute \(f'(c)\) for \(f(x) = x^3\) from the limit definition.
    Solution \(\frac{(c+h)^3 - c^3}{h} = \frac{3c^2h + 3ch^2 + h^3}{h} = 3c^2 + 3ch + h^2 \xrightarrow{h \to 0} 3c^2\).
  2. Is \(f(x) = x|x|\) differentiable at 0?
    Solution Yes! \(\frac{f(h) - 0}{h} = \frac{h|h|}{h} = |h| \to 0\). So \(f'(0) = 0\) — the two parabola halves \(x^2\) and \(-x^2\) meet smoothly, unlike the |x| corner.
  3. Define \(f(0) = 0\) and \(f(x) = x\sin(1/x)\) otherwise; also \(g(0) = 0\), \(g(x) = x^2\sin(1/x)\). Which is differentiable at 0?
    Solution For \(f\): \(\frac{f(h)}{h} = \sin(1/h)\) oscillates forever — no limit, not differentiable (though continuous!). For \(g\): \(\frac{g(h)}{h} = h\sin(1/h) \to 0\) by squeeze (\(|h\sin(1/h)| \le |h|\)) — differentiable with \(g'(0) = 0\).
  4. Find \(a, b\) so that \(f(x) = x^2\) for \(x \le 1\) and \(f(x) = ax + b\) for \(x > 1\) is differentiable everywhere.
    Solution Match values at 1: \(a + b = 1\). Match slopes: \(a = 2\cdot1 = 2\), hence \(b = -1\). So \(f(x) = 2x - 1\) for \(x>1\) — the tangent line to the parabola at (1,1).
  5. (Fermat) Show: if \(f\) has an interior maximum at \(c\) and \(f'(c)\) exists, then \(f'(c) = 0\).
    Solution For \(h > 0\): \(\frac{f(c+h)-f(c)}{h} \le 0\) (numerator ≤ 0), so the right limit \(\le 0\). For \(h < 0\): the quotient \(\ge 0\), so the left limit \(\ge 0\). Both equal \(f'(c)\), forcing \(f'(c) = 0\). ∎

Convex and Concave Functions

Intuition: A convex function is bowl-shaped: connect any two points of the graph with a straight chord, and the graph sags below the chord. Concave is the opposite (dome-shaped). This one geometric property is the backbone of modern optimization.

Formal definition: \(f\) is convex if for all \(x, y\) and all \(\lambda \in [0, 1]\):

\[f(\lambda x + (1-\lambda)y) \;\leq\; \lambda f(x) + (1-\lambda)f(y)\]

The left side is the graph at a mixed point; the right side is the chord at the same place. Concave flips the inequality. \(f\) is concave ⟺ \(-f\) is convex.

Interactive plot: pick a function and chord endpoints \(a, b\); the shaded region is the gap between chord and curve. The λ-slider walks a point along the chord and compares both sides of the definition numerically.

Step-by-step: \(f(x) = x^2\) is convex, straight from the definition

Step 1: Write both sides for \(f(x) = x^2\)
Need: \((\lambda x + (1-\lambda)y)^2 \le \lambda x^2 + (1-\lambda)y^2\).

Step 2: Expand the left side \[\lambda^2 x^2 + 2\lambda(1-\lambda)xy + (1-\lambda)^2 y^2\] Step 3: Subtract left from right
\[\lambda(1-\lambda)x^2 - 2\lambda(1-\lambda)xy + \lambda(1-\lambda)y^2 = \lambda(1-\lambda)(x - y)^2\] Step 4: Conclude
\(\lambda \in [0,1]\) makes \(\lambda(1-\lambda) \ge 0\), and \((x-y)^2 \ge 0\) always — so the difference is \(\ge 0\), proving the inequality. ∎
Second-derivative test & inflection points: for twice-differentiable \(f\): \(f'' \ge 0\) on an interval ⟺ convex there; \(f'' \le 0\) ⟺ concave. An inflection point is where convexity flips.

Worked: \(f(x) = x^3 - 3x\). \(f''(x) = 6x\): negative for \(x < 0\) (concave), positive for \(x > 0\) (convex) → inflection at \(x = 0\). Try it in the widget with \(x^3/8\): place \(a, b\) straddling 0 and watch the chord cross the curve.
Real-World Applications:
  • Convex optimization: for a convex function, any local minimum is automatically global — the bowl has one bottom. This is why machine-learning practitioners prize convex losses; least squares \(\sum (y_i - \hat y_i)^2\) is convex.
  • Jensen's inequality: for convex \(f\): \(f(\mathbb{E}[X]) \le \mathbb{E}[f(X)]\) — the definition applied to weighted averages; foundational in probability and information theory.
  • Economics: diminishing returns = concave production functions; risk aversion = concave utility.

Exercises:

  1. Show \(f(x) = e^x\) is convex using the second-derivative test.
    Solution \(f''(x) = e^x > 0\) for all \(x\) → convex on all of \(\mathbb{R}\).
  2. Find the intervals of convexity/concavity of \(f(x) = x^4 - 6x^2\).
    Solution \(f''(x) = 12x^2 - 12 = 12(x-1)(x+1)\): convex on \((-\infty,-1]\) and \([1,\infty)\), concave on \([-1,1]\); inflections at \(x = \pm 1\).

Research Level

Distributions and Generalized Functions

Distributions (in the sense of Schwartz) extend functions, allowing "functions" like the Dirac delta.

Operator Theory

Operators are functions between function spaces. A linear operator \(T: V \rightarrow W\) satisfies:

\[T(\alpha f + \beta g) = \alpha T(f) + \beta T(g)\]

Eigenvalue Problem

For an operator \(T\):

\[T(f) = \lambda f\]

\(\lambda\) is an eigenvalue and \(f\) is an eigenfunction.

Holomorphic Functions (Complex Analysis)

A function \(f: \mathbb{C} \rightarrow \mathbb{C}\) is holomorphic (or analytic) if it is complex differentiable.

Cauchy-Riemann Equations: If \(f(x + iy) = u(x, y) + iv(x, y)\), then \(f\) is holomorphic if:

\[\frac{\partial u}{\partial x} = \frac{\partial v}{\partial y}, \quad \frac{\partial u}{\partial y} = -\frac{\partial v}{\partial x}\]
