Beginner Level

Equations

What is an Equation?

An equation is a mathematical statement asserting that two expressions are equal. The symbol \(=\) indicates equality:

\[\text{Left Side} = \text{Right Side}\]

Linear Equations in One Variable

A linear equation in one variable has the form:

\[ax + b = c\]

where \(a \neq 0\).

Example: Solve \(2x + 3 = 7\)
\[2x = 4 \Rightarrow x = 2\]

Quadratic Equations

A quadratic equation has the form:

\[ax^2 + bx + c = 0\]

The quadratic formula gives solutions:

\[x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\]

Inequalities

What is an Inequality?

An inequality is a mathematical statement comparing two expressions. Inequalities use relation symbols:

  • \(>\) (greater than)
  • \(<\) (less than)
  • \(\geq\) (greater than or equal to)
  • \(\leq\) (less than or equal to)

Linear Inequalities

Similar to equations, but with one key difference:

When multiplying or dividing by a negative number, flip the inequality sign.

Example: Solve \(-3x + 5 < 11\)
\[-3x < 6 \Rightarrow x > -2\] (inequality flipped!)

Absolute Value Inequalities

\(|x| < a \iff -a < x < a\) (where \(a > 0\))
\(|x| > a \iff x < -a \text{ or } x > a\) (where \(a > 0\))

Intermediate Level

Equations

Systems of Linear Equations

Multiple linear equations with multiple variables can be solved using substitution or elimination methods.

Example: \[\begin{cases} x + y = 5 \\ 2x - y = 4 \end{cases}\] Solution: \(x = 3, y = 2\)

Polynomial Equations

A polynomial equation of degree \(n\) has the form:

\[a_nx^n + a_{n-1}x^{n-1} + ... + a_1x + a_0 = 0\]

Exponential & Logarithmic Equations

Exponential: \(a^x = b \Rightarrow x = \frac{\log(b)}{\log(a)}\)

Logarithmic: \(\log_b(x) = c \Rightarrow x = b^c\)

Inequalities

Quadratic Inequalities

A quadratic inequality like \(ax^2 + bx + c > 0\) is solved by:

  1. Finding roots of the equation
  2. Testing the sign in each interval
Example: \(x^2 - 5x + 6 > 0\)
Roots: \(x = 2, x = 3\)
Solution: \((-\infty, 2) \cup (3, \infty)\)

Rational Inequalities

For \(\frac{p(x)}{q(x)} > 0\), find critical points where numerator = 0 or denominator = 0, then test sign in each region.

Classical Inequalities

  • AM-GM: \(\frac{a_1 + ... + a_n}{n} \geq \sqrt[n]{a_1 \cdot ... \cdot a_n}\)
  • Cauchy-Schwarz: \((a_1b_1 + ... + a_nb_n)^2 \leq (a_1^2 + ... + a_n^2)(b_1^2 + ... + b_n^2)\)
  • Triangle: \(|a + b| \leq |a| + |b|\)

Advanced Level

Equations

Algebraic Equations and Field Extensions

An algebraic equation is studied through field extensions and Galois theory.

Abel-Ruffini Theorem: Polynomials of degree 5 or higher cannot generally be solved by radicals.

Diophantine Equations

These are polynomial equations where we seek integer or rational solutions.

Example: \(ax + by = c\) has integer solutions if and only if \(\gcd(a, b) \mid c\).

Fermat's Last Theorem

The equation \(x^n + y^n = z^n\) has no positive integer solutions for \(n \geq 3\).

Proved by Andrew Wiles in 1995, solving a 358-year-old conjecture.

Inequalities

Real Analysis Inequalities

  • Hölder's Inequality: \(\sum |a_i b_i| \leq \left(\sum |a_i|^p\right)^{1/p} \left(\sum |b_i|^q\right)^{1/q}\) where \(\frac{1}{p} + \frac{1}{q} = 1\)
  • Minkowski's Inequality: \(\left(\sum |a_i + b_i|^p\right)^{1/p} \leq \left(\sum |a_i|^p\right)^{1/p} + \left(\sum |b_i|^p\right)^{1/p}\)

Partial Orders and Lattices

A relation \(\leq\) is a partial order if it is reflexive, antisymmetric, and transitive.

Operator Inequalities

For bounded linear operators \(A, B\) on a Hilbert space:

\[\|A + B\| \leq \|A\| + \|B\|\]

Research Level

Equations

Elliptic Curves

An elliptic curve is defined by:

\[y^2 = x^3 + ax + b\]

These form an abelian group and are fundamental in number theory, algebraic geometry, cryptography, and the proof of Fermat's Last Theorem.

Modular Forms and Taniyama-Shimura Conjecture

The Taniyama-Shimura-Weil conjecture (proved by Wiles) connects elliptic curves to modular forms, providing the key insight for proving Fermat's Last Theorem.

Transcendental Number Theory

Lindemann-Weierstrass Theorem: If \(\alpha_1, ..., \alpha_n\) are algebraic numbers linearly independent over \(\mathbb{Q}\), then \(e^{\alpha_1}, ..., e^{\alpha_n}\) are algebraically independent over \(\mathbb{Q}\).

This proves \(\pi\) and \(e\) are transcendental.

Inequalities

Variational Inequalities

Seek \(u \in K\) such that:

\[\langle Au - f, v - u \rangle \geq 0 \quad \forall v \in K\]

Applications: contact problems, obstacle problems, optimal control, complementarity problems.

Integral Inequalities

  • Jensen's Inequality: For convex \(\phi\): \(\phi\left(\frac{1}{b-a}\int_a^b f\,dx\right) \leq \frac{1}{b-a}\int_a^b \phi(f)\,dx\)
  • Grönwall's Inequality: If \(u(t) \leq f(t) + \int_0^t c(s)u(s)\,ds\), then \(u(t) \leq f(t)e^{\int_0^t c(s)\,ds}\)

Matrix Inequalities

Löwner Partial Order: For Hermitian matrices, \(A \leq B\) if \(B - A\) is positive semidefinite.
